# 🌐 My Portfolio

A personal portfolio website showcasing my projects, services, and skills as a frontend developer.

---

## 🚀 Features

* 📱 Mobile responsive design
* 🎨 Clean dark-themed UI
* ⚡ Smooth animations with proper duration
* 🧩 Sections included:

  * Home
  * Services
  * Projects
  * About Me
  * Contact

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Tailwind CSS
* ⚡ Vite
* 💻 HTML5
* 🎯 JavaScript

---

## 📌 Project Status

* Tablet & Desktop responsiveness 🔄 Ongoing
